# 这是一个AI导航网站
使用的技术战为 React+Express

## mac运行
### 安装nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.zshrc

### 安装node

安装最新的 LTS (长期支持) 版本：
LTS 版本通常更稳定，推荐用于生产环境。

nvm install --lts

### 测试
进入项目根目录  
npm install  
npm run dev
### 打包运行
npm run build  
npm start
